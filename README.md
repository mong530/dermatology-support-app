\# Dermatology Support App（皮膚科 症状検索・説明支援アプリ）



\## 概要

皮膚科外来での「症状→候補疾患の整理」と「患者説明」を補助するアプリです。  

診断確定ツールではなく、医師の判断を前提にした説明支援として設計しています。



\## 主な機能

\- 症状・病名の検索/表示（data.csvベース）

\- 別名（aliases）による検索揺れ吸収

\- 鑑別ポイント・初期対応の提示

\- 運用を想定したデータ更新（管理者Excel → CSV変換）

\- 入力ミス防止（必須チェック・フラグ値バリデーション・バックアップ自動作成）



\## データ更新フロー（初心者運用を想定）

1\. 管理者が `tools/admin\_edit\_template.xlsx` を編集

2\. `tools/CSV出力.bat` をダブルクリック

3\. `derma\_app/data.csv` が更新され、バックアップが自動作成される
