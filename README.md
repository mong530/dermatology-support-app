# 皮膚科症状説明アプリ

## 概要

皮膚科診察時の説明のばらつきや診察時間の不足といった課題を想定し、
医師が短時間で症状や治療法を患者に説明できるようにすることを目的としたアプリです。
PythonとStreamlitを用いて開発しました。

## 主な機能

* 症状検索機能
* サジェスト検索（1文字入力で候補を最大8件表示）
* 頻出病名ボタン（よく使う病名をホーム画面に表示）
* 一覧表示 → クリックで症状・治療法の詳細表示
* 治療法のメリット／デメリットを箇条書き表示
* CSVデータによる症状情報管理
* 管理者用テンプレートによるデータ更新
* Excelからdata.csvを生成するツール
* 入力漏れチェック機能

## 使用技術

* Python
* Streamlit
* Pandas
* CSVデータ管理

## アプリURL

https://dermatology-support-app.streamlit.app/

## 開発背景

皮膚科では診察時間が限られており、症状や治療法の説明が医師によってばらつく可能性があると考えました。
そこで、検索機能や頻出病名ボタンを用いて短時間で情報を提示できるUIを設計しました。
